# QuickChart Gallery

Zenbench generates [QuickChart](https://quickchart.io) URLs for embedding benchmark
charts in READMEs. No API key needed — Chart.js config is URL-encoded inline.

## Color schemes

Six built-in schemes via `ColorScheme::phosphor()`, `ocean()`, `ember()`, `arctic()`, `sunset()`, `mono()`.

```rust
use zenbench::quickchart::{QuickChartConfig, ColorScheme};

// Use a preset
let config = QuickChartConfig::with_scheme(ColorScheme::ocean());

// Or customize the default
let config = QuickChartConfig {
    colors: vec![("mozjpeg".into(), "#ff9800".into())],
    ..Default::default() // phosphor scheme
};

let urls = suite.to_quickchart_urls(&config);
let markdown = suite.to_quickchart_markdown(&config);
```

|---|---|---|

| `padding` | 58 | Fixed vertical padding (title + axes) |

| `colors` | `[]` | `Vec<(pattern, hex)>` for per-benchmark name matching |
